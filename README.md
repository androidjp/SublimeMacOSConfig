# SublimeMacOSConfig
Sublime configure on MacOS
## 一、配置相关
1. Sublime 用户配置信息存放目录: ~/Library/Application\ Support/Sublime\ Text\ 3/Packages/User/
2. 可以将自己的配置保存到github中，必要时拉取到此目录：
  * cd ~/Library/Application\ Support/Sublime\ Text\ 3/Packages/
  * ls
  * rm -rf User
  * git clone xxxxxxxx
  * mv xxxxx User
